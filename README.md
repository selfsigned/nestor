# GO Nestor 
A  __command line__ [nestor](https://nestorparis.com/) client written in go.
![demo](./assets/nestor.png)

## Usage
``` bash
$ ./nestor
$ ./nestor -week
$ ./nestor --help
```
## Installation
### From source
``` bash
$ go get -d .
$ go build
```
